# Markdown Note

## 结构
- `nodejs-server`  API
- `react-client`   客户端


## 运行

**Clone**
```shell script
git clone https://github.com/diandianzd/markdown-note
```

**客户端 code**
```shell script
cd react-client && npm install
npm start
```

**API code**
```shell script
cd nodejs-server && npm install
npm start
```
