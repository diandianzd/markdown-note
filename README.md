# Markdown Note

## 结构
- `nodejs-server`  nodejs api
- `react-client`  react client 
- ~~`vue-client`  vue client~~


## 运行

**Clone**
```shell script
git clone https://github.com/diandianzd/markdown-note
```

**Client code**
```shell script
cd react-client && npm install
npm start
```

**Server code**
```shell script
cd nodejs-server && npm install
npm start
```
